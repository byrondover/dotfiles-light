# Dotfiles Light

Light, deployable dotfiles. v0.4.

Alternative to Bryan Stitt / Ryan Bates originals.

## Usage

### Install dotfiles.

```shell
./install.sh
```

### Gather dotfiles.

```shell
./gather.sh
```
